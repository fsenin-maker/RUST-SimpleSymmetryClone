# RUST-SimpleSymmetryClone
Симметричное строительство с красивым UI, авто-детектом, поддержкой дверей и апгрейда
Как загрузить плагин на сервер, чтобы он точно работал

Убедись, что на сервере установлен uMod (Oxide) последней версии (скачивается с umod.org).
Зайди в папку сервера → oxide → plugins.
Создай новый файл с именем SimpleSymmetryClone.cs и вставь туда весь код выше.
Сохрани файл (кодировка UTF-8 без BOM, если редактор спрашивает).
Перезапусти сервер или в консоли сервера выполни команду: oxide.reload SimpleSymmetryClone (если плагин уже был) или просто oxide.load * для загрузки всех новых.
В консоли сервера должно появиться сообщение вроде:
Loaded plugin SimpleSymmetryClone v1.2.1 by GrokAI

Если не загрузился — проверь логи (oxide/logs) на ошибки компиляции.
3. Все команды и разрешения
Разрешение (permission):
simplesymmetryclone.use
Как выдать:

Всем по умолчанию: в консоли сервера oxide.grant group default simplesymmetryclone.use
Только определённой группе (например, vip): oxide.grant group vip simplesymmetryclone.use
Конкретному игроку: oxide.grant user ИмяИгрока simplesymmetryclone.use

Команды в чате (все через /sym):

/sym — показывает справку по командам и доступным типам.
/sym toggle — включает или выключает симметрию.
/sym set — устанавливает центр симметрии (смотри на пол/фундамент и пиши команду).
/sym set auto — устанавливает центр + сразу автоматически определяет тип базы (n2s/n3s/n4s/n6s).
/sym auto — только автоопределение типа (центр уже должен быть установлен).
/sym delete — удаляет центр симметрии.
/sym show — показывает координаты центра и текущий тип.
/sym ui — переключает UI (в этом клоне только заглушка, ничего не показывает).
/sym n2s / /sym n3s / /sym n4s / /sym n6s / /sym m2s / /sym m4s — вручную выбирает тип симметрии.

Типы симметрии:

n2s, n3s, n4s, n6s — ротационная (поворотная) симметрия.
m2s, m4s — зеркальная.

Рекомендуемый порядок использования:

Построй основу базы (фундаменты).
Встань в центр, посмотри вниз → /sym set auto (центр + автоопределение типа).
/sym toggle (включи симметрию).
Строи дальше — всё будет дублироваться симметрично, включая двери, бойницы и апгрейды.
